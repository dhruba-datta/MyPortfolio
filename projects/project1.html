EasyCooking :

Project information

  • Category: Web Development
  • Project date: May, 2021
  • GitHub Link: GitHub
  • Project URL: www.EasyCooking.com

EasyCooking is a cooking website from where you will find different dishes based on different cuisine! currently 3 cuisines are available there: Bengali, Indian & Chinese. You can find different dishes and different cooking methods on them. In description you'll find the ingredient details. Also you can buy products from the 'Shop Now' button on navigation.
